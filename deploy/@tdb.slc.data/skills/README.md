# Module Skills

This folder is for skills that are local to `@tdb/slc-data`.

Canonical system skills live in:

- `https://github.com/sys-repo/sys.canon`

Canonical doctrine lives in:

- `https://github.com/sys-repo/sys.canon`

Use this folder for:

- skills that are intentionally local to `@tdb/slc-data`
- module workflow/operator skills over the `@tdb/slc-data` CLI
- candidate skills being tested here before possible promotion to canon

Do not duplicate canonical skills here unless a module-local variant is explicitly needed.

When a local skill becomes stable, broadly reusable, and no longer depends on `@tdb/slc-data` workflow assumptions, consider promoting it into canon.
