# @sys/tools

CLI surface for @sys primitive compositions.

```bash
deno run -A jsr:@sys/tools
deno run -A jsr:@sys/tools --help
```

Automatic root update advisory checks can be disabled with:

```bash
deno run -A jsr:@sys/tools --no-update-check
SYS_TOOLS_NO_UPDATE_CHECK=1 deno run -A jsr:@sys/tools
```
