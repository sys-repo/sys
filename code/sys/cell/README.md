# @sys/cell

A **Cell** is a folder-shaped [metamedium](https://en.wikipedia.org/wiki/Metamedia) whose [DSL](https://martinfowler.com/dsl.html) stores meaning and whose meaning can be interpreted, viewed, and validly rewritten within the folder that bounds it.

The medium is ordinary files in a folder: Markdown, YAML, TypeScript, JSON, binary data, and other file-carried forms.

```text
@sys/cell
 - runtime
 - dsl
 - view

Runtime = functional environment
DSL     = stored meaning
View    = bound perception
```
