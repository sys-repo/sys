# Repo Skills

This folder is for skills that are local to this system workspace.

Canonical system skills and doctrine live in:

- `https://github.com/sys-repo/sys.canon`

Use this folder for:

- skills that are intentionally local to the generated repo
- repo workflow/operator skills for that workspace
- candidate skills being tested locally before possible promotion to canon

Do not duplicate canonical skills here unless a repo-local variant is explicitly needed.

When a local skill becomes stable, broadly reusable, and no longer depends on repo-specific workflow assumptions, consider promoting it into canon.
